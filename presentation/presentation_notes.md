# Presentation Requirements
- 7-15 minutes
- own graphics and performance testing

Slides:
- title slide: name, affiliation, name of talk, class, and date
  - should also be the last slide
- state the problem
- prototypes and variables slide
  - class name
  - method
  - parameters
  - return value
- constraints slide
- solution slides
- performance slides
  - running time
  - performance chart
- topcoder statistics slide
  - opened the problem
  - submitted the problem
  - correct submissions
  - success rate
  - best time
  - average correct time
